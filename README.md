# 🇰🇷 korea-trip-tsp (대한민국 경로 최적화 서비스)

한국 내 차량 여행 시, 사용자가 입력한 장소 리스트를 기반으로 **최적의 방문 순서를 계산해주는 Python 기반 AI 경로 최적화 프로젝트**입니다. 

사용자가 방문하고 싶은 장소와 숙소 정보를 입력하면, AI가 실행 가능한 **최적의 여행 일정을 자동으로 생성**하고, 실제 도로망을 기준으로 이동 효율이 가장 높은 일정을 제안 및 시각화해 주는 **지능형 여행 계획 서비스**입니다.

---

## 🗺️ 프로젝트 소개

기존의 국내 여행 서비스들은 단순한 장소 추천이거나, 점과 점 사이의 단순 거리 계산에 그치는 경우가 많습니다. 

**korea-trip-tsp**는 사용자가 직접 입력한 장소들을 기반으로, 단순히 최단 거리를 구하는 것을 넘어 **"어떤 장소들을 어떤 날에 함께 묶어서 방문하는 것이 가장 합리적이고 편안한 동선인가?"**라는 근본적인 외판원 문제(TSP)를 해결하는 데 초점을 맞췄습니다.

여행자의 숙박 패턴(단순 이동, 연박 등)을 AI가 지능적으로 분석 및 그룹화(Clustering)하여 가장 현실적이고 쾌적한 드라이브 코스를 제안합니다.

---

## ✅ 주요 기능 및 특징

### 📍 하이브리드 경로 최적화 (TSP 기반)
단순 직선거리가 아닌 **OSMnx**를 활용한 실제 도로 기반 실거리를 계산합니다. 일정의 성격에 따라 두 가지 알고리즘을 교차 적용합니다.
* **[이동일] Vector Projection:** 출발지와 도착지가 다를 경우, 직진 방향을 거스르는 '역주행'을 원천 차단하기 위해 방향 벡터를 기준으로 목적지를 1차 정렬합니다.
* **[연박일] Sweep Algorithm (각도 정렬):** 같은 숙소로 되돌아올 경우, 지그재그 교차 동선(모래시계 현상)을 막기 위해 숙소를 중심으로 목적지들을 시계방향(각도)으로 쓸어 담아 완벽한 순환 드라이브 코스를 만듭니다.
* **[공통] 2-Opt 알고리즘:** 1차 정렬된 동선에서 발생할 수 있는 교차로(X자 꼬임)를 찾아내어 일자로 풀어줍니다.

### 🗓️ 지능형 일정 자동 분할 (K-Means Clustering)
* **스마트 연박(Basecamp) 감지:** 사용자가 1박씩 숙소를 옮기든, 특정 숙소에 며칠 머무르든 하드코딩 없이 체류 패턴을 분석하여 자동으로 날짜 그룹을 생성합니다.
* 며칠에 걸친 연박 일정의 경우, **K-Means 클러스터링**을 통해 목적지들을 지리적으로 뭉쳐서(예: 1일차는 동쪽, 2일차는 서쪽 투어) 날짜별로 균등하게 쪼개줍니다.

### 🔎 카카오맵 API 시각화
* 도출된 최적 경로를 카카오맵 위에 렌더링하며, 날짜별 탭 기능을 통해 특정 날짜의 경로와 마커만 직관적으로 확인할 수 있습니다.

## 🚀 트러블슈팅 (문제 해결 과정)

**1. 연박 일정 시 동선이 지그재그로 꼬이는 현상 해결**
* **문제:** 출발지와 도착지가 같은 연박 일정에서 '가장 가까운 목적지' 순으로만 경로를 탐색했더니, 거리가 가깝다는 이유로 지도를 가로지르는 교차 동선이 발생함.
* **해결:** 연박 일정의 경우 거리 기준이 아닌, 숙소를 중심으로 한 **'각도(시계 방향)'를 기준으로 목적지들을 우선 정렬**한 뒤 세부 최적화를 진행하여, 한 방향으로 둥글게 도는 깔끔한 드라이브 코스로 개선함.

**2. 체크아웃(마지막) 날짜의 일정 누락 현상 해결**
* **문제:** "해당 날짜에 잠을 자는 숙소"를 기준으로 날짜를 그룹핑하다 보니, 숙박 없이 체크아웃 후 이동만 하는 마지막 날이 일정 분할 로직에서 누락되는 현상 발생.
* **해결:** 단순히 잘 곳만 확인하는 것이 아니라, "오늘 아침에 체크아웃한 숙소"까지 역추적하여 교차 검증하는 로직을 추가. 부분 연박 일정에서도 마지막 날까지 공평하게 목적지가 배분되도록 예외 상황(Edge Case) 처리.
